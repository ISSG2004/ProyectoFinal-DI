Descripción

Esta es una aplicación de agenda de contactos desarrollada en JavaFX, con almacenamiento en archivos JSON y múltiples opciones para visualizar los datos.

Funcionalidades

  • Listado de contactos con columnas de nombre y apellido.
  • Botones para agregar, editar y eliminar contactos.
  • Almacenamiento y carga de contactos en formato JSON.

Menú

  • File: New, Open, Save, Save As, Exit.
  • Edit: Opciones de edición de contactos.
  • Charts: PieChart, LineChart, BarChart, StackedAreaChart, Ejercicio A (gráfico de cumpleaños).
  • Help: Documentación de la aplicación.

Gráficos

La aplicación permite visualizar datos mediante diferentes tipos de gráficos:

  • PieChart
  • LineChart
  • BarChart
  • StackedAreaChart
  • Ejercicio A: PieChart basado en los cumpleaños de los contactos.

Instalación y Ejecución

git clone https://github.com/ISSG2004/ProyectoFinal-DI.git
mvn clean javafx:run
